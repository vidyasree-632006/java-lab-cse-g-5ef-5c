## using string buffer to delete,remove character 
source code 
```
class Deletechar{
  public static void main(String [] args){
        StringBuffer  sb = new StringBuffer("JAVA PROGRAMMING");
        System.out.println(sb);
        sb.deleteCharAt(4);
        System.out.println(sb);
        sb.delete(0,4);
        System.out.println(sb);
  }
}
```
